# CLAUDE.md — Estudio Jurídico Vera Constanzo

## Contexto del proyecto

Sitio web para el **Estudio Jurídico Vera Constanzo**, especializado en Derecho de Seguros y Daños. El foco principal es el **turno digital** (sistema de reserva de turnos online).

Stack: **Next.js + Tailwind CSS + Prisma**. La página principal (`page.tsx`) ensambla los componentes: Header → Hero → Services → WhyUs → Team → Contact → Footer. El sistema de turnos vive en `/turnos` y el panel admin en `/admin`.
